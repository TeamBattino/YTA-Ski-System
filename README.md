# YTA-Ski-System
YT-Ads Ski system Monorepo

## Setup Guide

1. Install Bun
```bash
curl -fsSL https://bun.sh/install | bash
```


### Webapp

1. Enter the webapp directory
```bash
cd webapp
```

2. Install dependencies
```bash
bun i
```

3. Start the development server
```bash
bun dev
```
The dev server will be running on `http://localhost:3000`
